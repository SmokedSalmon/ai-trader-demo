## Quick Debug for Modern IDE

### VS Code
- Debug Terminal  
  Instead of normal terminals, launch the `JavaScript Debug Terminal` and `node <your script>`
- `F5` Launch  
  Setup the `/.vscode/launch.json` to define what `F5` Launches  
  Refer to [launch.json](../.vscode/launch.json) of this project  
  > Recommended
- Auto Attach  
  Config VSCode to auto attach debugger to all javascript upon run
